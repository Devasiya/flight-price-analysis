# Flight Price Prediction ✈️

## 📌 Overview
This project analyzes flight data and performs feature engineering to prepare for price prediction.

## 📊 Features
- Data Cleaning
- Feature Extraction (Date, Time, Duration)
- Handling Missing Values
- Exploratory Data Analysis (EDA)

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

## 📁 Dataset
- Data_Train.xlsx
- Test_set.xlsx

## 🚀 Future Work
- Model Building (Regression)
